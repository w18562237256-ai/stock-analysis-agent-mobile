# 股票分析 Agent 手机版部署包

这是一个微信浏览器友好的纯静态网页版本，入口文件是 `index.html`。不需要服务器、不需要安装依赖，上传到静态网站平台即可访问。

页面已做手机优先优化：顶部模式切换吸顶、自选股横向滑动、个股分析单列展示、基本面/ST风险和中长期判断适配窄屏。

## 推荐部署方式

### Cloudflare Pages

1. 登录 Cloudflare Pages。
2. 选择上传静态文件。
3. 上传本文件夹里的全部内容，或直接上传压缩包。
4. 构建命令留空。
5. 输出目录填 `/` 或留空。

### Vercel

1. 新建项目。
2. 上传本文件夹，或把本文件夹推到 GitHub 后导入。
3. Framework Preset 选择 `Other`。
4. Build Command 留空。
5. Output Directory 留空。

### Netlify

1. 选择 Add new site。
2. 拖拽本文件夹到部署区域。
3. 发布后即可得到访问链接。

## 行情说明

页面使用东方财富公开行情接口，通过 JSONP 拉取实时数据。该接口不是官方承诺的稳定开放 API，后续可能变化；页面已设置失败回退提示。

## 风险提示

本工具只做行情和策略辅助分析，不构成投资建议。ST/退市风险识别基于名称标签、价格成交异常、市值和估值等公开字段做提醒，不能替代公告、财报和交易所正式信息。
